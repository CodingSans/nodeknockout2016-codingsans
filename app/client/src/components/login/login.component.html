Destructible messages as a service

Project made for Node Knockout 2016

We had 48 hours to create something using Node.JS
The project would have been a destructive message service, where you are able to create messages which are alive for a given time only.
After the time is over, the message is getting destroyed and there is no way to see it again.
We wanted to make it possible to control the service from Slack with custom commands. However, we had no time to finish. More information at the repository of the project!
